# Modern Netflix Clone - Glassmorphism Edition

A premium, Netflix-inspired landing page built with React, Tailwind CSS v4, and Framer Motion.

## Features
- **Dynamic Movie Search**: Real-time search using the IMDB API.
- **Glassmorphism UI**: Frosted glass cards, blurred backgrounds, and sleek transitions.
- **Search History**: Stores recent searches in localStorage.
- **Live Weather**: Shows current weather based on user location (Open-Meteo).
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Hero Section**: Featured movie with dynamic overlay and animations.
- **Movie Rows**: Netflix-style horizontal scrolling categories.

## Tech Stack
- **React 19** (Functional Components + Hooks)
- **Tailwind CSS v4** (Modern CSS-first approach)
- **Framer Motion** (Smooth animations and presence)
- **Lucide React** (Beautiful icon set)
- **Vite** (Next-gen frontend tooling)

## API Source
- Movie Data: `https://imdb.iamidiotareyoutoo.com`
- Weather: `https://api.open-meteo.com`
- Geocoding: `https://nominatim.openstreetmap.org`

## Getting Started
1. `npm install`
2. `npm run dev`
